Retr3d Release Notes 0.1.0

English Not Your First Language?

What is Retr3d?

Retr3d is a framework dedicated to affordable 3D printing equipment for developing economies that can be locally sourced, locally maintained and locally improved.

We believe that 3D printing can be as transformative in developing countries as the mobile phone. As with the mobile phone, which has already changed the way people across the Africa communicate, introducing 3D printing at the community level offers the potential to localize manufacturing. Which is why we are making it possible to build Retr3d machines from the thousands of tonnes of e-waste which would otherwise end up as landfill.

What's new in 0.1.0?

Everything.

With this being our first official release, everything is new. However, if you happened to use the prerelease Retr3d, the following are some features that are “new” in v0.1.0.

Configuration Generator

Gone are the days of editing configuration files by hand. In this new age we use the Configuration Generator which generates the needed files through an easy to use web interface. While you can still edit the files directly, we feel that this new interface is more user­friendly (and aesthetically appealing.)

Plater

To facilitate an efficient work flow, Retr3d can interface with ​Plater​ to arrange parts in an extremely efficient manner. This helps put more parts on one build plate to help print all of those tiny parts in a straight forward manner. Be as it some may still want to print parts individually, we still generate individual parts.

Slic3r Integration

Retr3d can now interface with ​Slic3r​ to slice parts automatically. This makes it possible for you go from computer to printer as fast as a jiff. Just don't run too fast!

Now Where Did that Part Go?

Retr3d generates a part checklist along with the parts, now you will never having an excuse for not having a Left Print Bed Bushing Holder on you at all times.

We're on a 1:1 Basis

To speed up production, Retr3d generates 1:1 printouts of every generated sheet metal part. You can print drawings out on a 2D printer, and not soon later, be printing things out on a 3D printer.

Zipp'n Up

Retr3d zips up STL files so you can keep all those important files safe from ninjas, or even give them away with a printer. With this, customers (or sneaky ninjas) can print replacement parts for their own printers, freeing you up to tighten your ninja security.
