Retr3d Release Notes 0.1.0

For English As A Second Language

What is Retr3d?

In 2009, discarded TVs, computers, peripherals (including printers, scanners, fax machines) mice, keyboards, and cell phones totaled about 2.37 million short tons. Most of our e­waste ends up in landfills—mostly in the developing world—where toxic metals leach into the environment. Retr3d uses python to 3D model printable parts for the construction of 3D printers from this growing e­waste.

What's new in 0.1.0?

Everything.

With this being our first official release, everything is new. However, if you happened to use the prerelease Retr3d, the following are some features that are “new” in v0.1.0.

Configuration Generator

The new Configuration Generator generates needed files through an easy to use web interface. While you can still edit the files directly, we feel that this new interface is more user friendly.

Plater

To facilitate an efficient work flow, Retr3d can interface with ​Plater​ to arrange parts in an extremely efficient manner. This helps put more parts on one build plate to help print all of those tiny parts in a straight forward manner. Be as it some may still want to print parts individually, we still generate individual parts.

Slic3r Integration

Retr3d can now interface with ​Slic3r​ to slice parts automatically. This makes it possible for you go from computer to printer faster then you could before.

Part Checklist

Retr3d generates a part checklist along with the parts, so you have an easy way to keep track of the may parts that go into making a printer.

1:1 Renderings

To speed up production, Retr3d generates 1:1 printouts of every generated sheet metal part. You can print drawings out on a 2D printer, and not soon later, be printing things out on a 3D printer.

Automatic Compression

Retr3d zips up STL files so you can keep all those important files together, in the hopes that you might give them away with a printer. With this, customers can print replacement parts for their own printers, freeing you up to do all sorts of things.
